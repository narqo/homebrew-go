# brew-go

## Install

~~~
% brew install brew-go
~~~

## Usage

~~~
% brew go install github.com/google/pprof
~~~

To update the package:

~~~
% brew go update github.com/google/pprof
~~~

To uninstall the package:

~~~
% brew go uninstall github.com/google/pprof
~~~
